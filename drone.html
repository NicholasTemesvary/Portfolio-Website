Overview

Drone-First-Search is an autonomous UAV platform designed for search-and-rescue missions, integrating computer vision, GPS navigation, and AI-based decision making.

Key Features

  • Real-time object detection and target acquisition using onboard vision systems.
  • GPS-integrated autonomous navigation with optimized search patterns.
  • Sensor fusion for improved reliability in dynamic environments.
  • Application of aerospace guidance, navigation, and control (GN&C) principles.

Technical Skills Applied

Python, OpenCV, AI/ML frameworks, flight control systems, UAV path planning, aerospace navigation protocols.
