Overview

This program allows for a the ability for images to be scaled down from any value into a compressed and downloadable file. Primarily allowed for personal use in Game-Jams but is open sourced and is meant to work on a multitude of image styles.

Key Features

  • Image compression and download
  • Allows for a multitude of file types including .PNG and .IMG
  • Scalable for mobile devices
  • Currently being developed to allow for video compression
